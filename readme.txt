Clone the repository by passing the git repos link in the text field 

Step 1:
cd AppFabricCloningFromForm

Step 2:
npm install express --no-bin-links

Step 3:
npm install body-parser --no-bin-links

Step 4:
node server/getFormData.js

Step 5:
open browser and type 
	
	To clone the repository
	http://localhost:8080/

	enter the url in the input field e.g:
	https://github.com/Ashok-Kumar-Venaktesh/Assignments.git
